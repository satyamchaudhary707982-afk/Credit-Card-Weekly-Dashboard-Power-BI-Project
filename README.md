# Credit-Card-Weekly-Dashboard-Power-BI-Project

📊 Credit Card Weekly Dashboard – Power BI Project
This project presents a comprehensive Credit Card Analytics Dashboard built in Power BI to deliver real-time insights into customer behavior, transaction trends, and revenue performance.

🚀 Project Objective
To develop a dynamic weekly dashboard that enables stakeholders to monitor key performance metrics, analyze trends, and make data-driven decisions for credit card operations.

📌 Key Features
💰 Revenue Analysis

Total Revenue, Interest Earned, and Transaction Amount tracking

Revenue breakdown by card category (Blue, Silver, Gold, Platinum)

📈 Transaction Insights

Weekly and quarterly transaction trends

Transaction count and volume analysis

👥 Customer Analytics

Revenue by Gender, Job Type, and Education Level

Income and salary group segmentation

Age group and dependent analysis

💳 Card Usage Behavior

Revenue by usage type (Swipe, Chip, Online)

Expense type analysis (Bills, Entertainment, Fuel, etc.)

📊 Interactive Filters

Quarter-wise filtering (Q1–Q4)

Week-level drill-down

Customer segmentation slicers

🛠️ Tools & Technologies
Power BI – Data Visualization & Dashboarding

DAX – Measures and calculated columns

Power Query – Data transformation & cleaning

Excel/CSV – Data source

📷 Dashboard Highlights
Credit Card Transaction Report (Revenue & Transactions Overview)

Customer Report (Demographics & Behavior Analysis)

Interactive visuals for deep insights

📊 Key Insights
Blue card category contributes the highest revenue

Swipe transactions dominate usage patterns

Business and white-collar customers generate significant revenue

Graduates form the largest revenue-generating segment

🎯 Conclusion
This dashboard helps businesses gain a 360° view of credit card operations, identify high-value customers, and optimize strategies for revenue growth and customer engagement.
